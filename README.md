# Tailflare UI

> Originally from https://github.com/aaanh/tailflare

Sync Tailscale hosts to Cloudflare subdomains.

# Production site

https://tailflare.aaanh.app

# Run it locally!

## Requirements

- Node >= 20.x

## Steps

```sh
git clone https://github.com/aaanh/tailflare-ui
```

```sh
cd tailflare-ui
```

```sh
npm install && npm run dev
```
